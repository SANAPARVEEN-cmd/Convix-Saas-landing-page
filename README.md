# 🚀 Convix Landing Page

## 📌 Overview
Convix is a modern **AI-powered marketing platform landing page** built for practice.  
It demonstrates how businesses can use AI tools to **automate campaigns, gain insights, and scale growth efficiently**.

This project focuses on **frontend development, UI/UX design, and SaaS product presentation**.

---

## 🎯 Features

- ✨ Clean and modern UI design  
- 🤖 AI-driven marketing concept  
- 📊 Analytics and dashboard showcase  
- 💬 Testimonials section  
- 💼 Creator marketplace section  
- 💳 Pricing plans (Free, Professional, Team)  
- 📢 Call-to-action for user engagement  

---

## 🧠 Purpose

This project was created to:

- Practice **React + Tailwind CSS**
- Improve **landing page design skills**
- Understand **SaaS product structure**
- Learn **conversion-focused UI and content**
